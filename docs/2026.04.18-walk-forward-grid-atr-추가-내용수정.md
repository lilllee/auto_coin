## 해석

### in-sample vs out-of-sample 차이

같은 전략의 730d in-sample 백테스트는 BTC +37.83% excess, XRP +80.84% excess 등 양호한 수치를 냈지만,
walk-forward 로 자르면 ±1% 안쪽으로 수렴. **이 gap 이 overfitting 의 크기다.**

### 파라미터 안정성

Window 별 best params 가 `cm=0.75 / 1.0 / 1.5 / 2.0` 전부 등장, 특정 regime 에 안정적으로 수렴하지 않음.
미래 1개월에 어떤 파라미터가 작동할지 예측 근거 없음.

### 시장별 sweet spot 불일치

2y in-sample 기준 최적 `channel_multiplier` 가
BTC → 1.5, XRP → 1.0, ETH → 2.0, SOL → 1.0 로 제각각.
단일 고정 설정으로 멀티마켓 운영 불가.

## 결론

- atr_channel_breakout 단독 판정: **STOP** — 소액 live 후보로 올릴 근거 없음.
- 현재 전략군 전체 판정: 6개 전략 모두 walk-forward 또는 trade count 기준 탈락.
  → 현 구조의 "일봉 + 가격 돌파 + 고정 파라미터" 틀 전체를 다시 설계해야 하는 상태.
